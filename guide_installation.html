Lance le script depuis ton terminal.

💻 Ouvrir un terminal :
Mac → Terminal.app ou iTerm2  ·  Windows → Windows Terminal
1 Connexion root — 1 seule fois clic pour copier
ssh root@IP_DU_VPS
⚠️ Si le terminal affiche Are you sure you want to continue? (yes/no) → tape yes puis Entrée.
📋 Valeurs demandées par le script
2 install-secure.sh — en root (~10-15 min) clic pour copier
bash <(curl -fsSL https://raw.githubusercontent.com/rockballslab/vps-secure/main/install-secure.sh)
📋 Valeurs demandées par le script
3 Connexion après installation — en vpsadmin clic pour copier
ssh vpsadmin@IP_DU_VPS -p 2222 -i ~/.ssh/id_ed25519_vps
Après reconnexion — vérification clic pour copier
sudo vps-secure-verify
vps-secure · github.com/rockballslab/vps-secure · Fichier local — aucune connexion réseau
🗑️

Tout effacer ?

Toutes les valeurs saisies seront supprimées localement. Cette action est irréversible.
